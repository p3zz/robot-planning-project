## Map drawer

### Setup

- Install dependencies:
```bash
sudo apt install python3-tk
```

```bash
pip3 install graphics.py
```

### Run

To visualize only the map:
```bash
cd disegnino
python3 main.py path/to/map  
```

To visualize the whole race:
```bash
cd disegnino
python3 main.py path/to/map path/to/moves  
```
